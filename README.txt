Installation: 

VOCAL is installed in C:\Program Files (x86)\vocal

The executable is: C:\Program Files (x86)\vocal\core\Calipso.exe


Upon running the executable, associated files will be written into a 
vocal directory in your default %APPDATA% path, which is probably:

C:\Users\[username]\AppData\Local\Roaming\



Uninstallation:

1. Navigate to the Control Panel
2. Select "Programs"
3. Select "Programs and Features"
4. Find "vocal version [number]", right-click, and select "uninstall."
