class PolygonDrawer(canvas, master, tag='', color='')[source]¶

Displays the polygon objects onto the canvas by supplying draw methods.

Parameters:
  • canvas – The canvas to draw the object to
  • master – Central class, in this case Calipso
  • tag (str) – Name of shape
  • color (str) – Color of shape
